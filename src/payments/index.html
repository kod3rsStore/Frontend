Ejemplo de envio de pago a Stripe

Datos Necesarios a cargar en el formulario

Metodo post del formulario: se debe enviar en la liga el id del usuario para obtener su carrito de compras

El campo data-amount en el formulario lleva el monto (este campo es visual ya que sale en la ventana de stripe pero el monto real lo calcula el backend en base al carrito de compras del usuario)

Se puede agregar un mejor logo en data-image para que se vea en el encabezado del form

el boton con clase stripe-button-el es el que trae el form pero para personalizar el boton se le dio un display none y se creo u n nuevo boton

la data-key es la llave que nos da stripe esta se puede/deberia ser manejada como variable de entorno
